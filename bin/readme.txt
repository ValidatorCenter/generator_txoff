Getting the RLP-command to stop the node:
1) Rename the file "_start.json" to "start.json"
2) Fill out the file "start.json"
3) Run the file "run_win.bat" - for Windows or "run_lin.sh" - for Linux

Получение RLP-команды для остановки ноды:
1) Переименуйте файл "_start.json" в "start.json"
2) Заполните файл "start.json"
3) Запустите файл "run_win.bat" - для Windows или "run_lin.sh" - для Linux
